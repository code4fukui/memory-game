# memory-game

A simple memory game built with JavaScript.

## Demo
You can play the game at https://code4fukui.github.io/memory-game/

## Features
- Memorize and click on numbers that appear on the screen
- Difficulty increases as the game progresses
- Fully client-side, no server required

## Usage
Open the `index.html` file in a web browser to play the game.
